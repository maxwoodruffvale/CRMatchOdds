Steps Page

How to get my API key

  1. Register for an account at the Clash Royale API
  2. Create a new key
  3. Set the IP address to be your current IP address
  4. Paste into the form; it will be saved automatically

How to get my tag

  1. Open Clash Royale on your phone
  2. Tap on your profile
  3. Copy down the letters after # below your name
